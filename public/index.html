Usage

  1. Print an ISO 12642-2 test chart (1617 patches)
  2. Scan unlaminated → export CGATS (.txt)
  3. Laminate, scan at least 1 strip (~25+ patches)
  4. Upload both files below
  5. Build DeviceLink → download .icc
  6. Apply in RIP or color server

Method: Frozen ResNet (pre-trained on reference printer) + Ridge(5→5) residual adapter. Predicts laminated spectra from unlaminated + ~25 anchor patches.

Reference: R2_11-4-23 (1617 pairs, P95=1.09).
